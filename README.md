## impacket-macos

在`macOS`上安装`impacket`工具包

## `impacket`工具包项目地址
```bash
https://github.com/SecureAuthCorp/impacket
```

### 一、使用方法

1. `git clone` 本项目
```bash
git clone https://github.com/imfiver/impacket-macos.git
```
2. 安装
```bash
cd impacket-macos
chmod +x install.sh
sudo ./install.sh
```
